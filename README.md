### Installation

Follow the instructions to clone this project

2. Clone the repo
   ```sh
   git clone https://github.com/erickctra/personal-website.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run
   ```js
   npm run dev
   ```
